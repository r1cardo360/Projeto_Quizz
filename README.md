# Quiz Naruto - Projeto de Treinamento de DOM com JavaScript

Este é um projeto simples de quiz sobre o anime **Naruto**, desenvolvido com HTML, CSS, Bootstrap e JavaScript. O objetivo do projeto foi treinar o uso do **DOM** e manipulação de eventos com JavaScript, aplicando o conhecimento adquirido durante o curso.

## Estrutura do Projeto

O projeto está organizado da seguinte forma:

```
/Quizz_Javascript
│
├── /CSS
│   └── style.css
|
├── /JS
│   └── script.js
│
└── index.html
```

### Funcionalidades

- **7 Perguntas sobre o anime Naruto**: Teste seus conhecimentos sobre a série com perguntas variadas.
- **Interatividade**: O quiz permite que o usuário selecione suas respostas, receba feedback e veja seu desempenho.
- **Design Responsivo**: Utilizando o CSS, o layout é adaptável a diferentes tamanhos de tela.

### Tecnologias Utilizadas

- **HTML**: Estruturação do conteúdo.
- **CSS**: Estilo do projeto.
- **JavaScript**: Manipulação do DOM e interatividade.

### Como Usar

1. Clone este repositório ou baixe o código.
2. Abra o arquivo `index.html` no seu navegador.
3. Teste seus conhecimentos sobre Naruto e veja quantas respostas você consegue acertar!
